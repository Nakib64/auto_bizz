# Auto Bizz Dashboard

A modern, responsive sales dashboard built with Next.js 14, TailwindCSS, shadcn/ui, and Recharts.

## Features

- **Modern Design**: Glassmorphism, soft shadows, and smooth animations.
- **Interactive Charts**: Visual sales data using Recharts.
- **Data Filtering**: Real-time filtering by date, price, and customer details.
- **Responsive**: Optimized for desktop and mobile devices.
- **Performance**: React Query for caching and efficient data fetching.

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: TailwindCSS v4
- **UI Library**: shadcn/ui
- **Charts**: Recharts
- **State Management**: React Query (TanStack Query)
- **Animations**: Framer Motion
- **Database**: MongoDB (Client ready)

## Getting Started

1.  Clone the repository.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Set up environment variables (optional for MongoDB):
    Copy `.env.example` to `.env.local` and add your MongoDB URI.
4.  Run the development server:
    ```bash
    npm run dev
    ```

## Deployment

This project is ready for deployment on Vercel.
